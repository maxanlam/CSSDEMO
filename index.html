04 — Bilder & object-fit

Alle Bilder haben dieselbe Größe (200×400px). Nur object-fit unterscheidet sich.

object-fit: cover
füllt den Rahmen, schneidet ab

object-fit: contain
passt rein, lässt Lücken

object-fit: fill
streckt das Bild (verzerrt)

object-fit: none
originalgröße, wird abgeschnitten

object-fit: scale-down
wie contain, aber nie größer als original

Basis für alle Bilder: img { max-width: 100%; display: block; }

17 — Details & Summary (CSS-only Interaktion)

Der Browser setzt open automatisch beim Klick. CSS reagiert mit details:open — kein JavaScript nötig.

Was ist CSS?

CSS (Cascading Style Sheets) bestimmt das Aussehen von HTML-Elementen — Farben, Abstände, Schriften, Layouts und vieles mehr.

Was ist das Box Model?

Jedes HTML-Element ist eine rechteckige Box aus Inhalt, Padding, Border und Margin.

Der hier ist bereits geöffnet

Mit dem Attribut open direkt im HTML startet es geöffnet. CSS reagiert mit details:open — das + dreht sich zum ×.


18 — Sonstiges: calc() & columns

calc() — verschiedene Einheiten kombinieren:

width: calc(100% - 80px)

columns — Fließtext automatisch aufteilen:

Das ist ein längerer Text der automatisch in drei Spalten aufgeteilt wird. CSS übernimmt den Umbruch komplett selbstständig. Man muss sich nur um column-count und column-gap kümmern. Das ist besonders praktisch für Zeitungs- oder Magazin-Layouts wo viel Text vorhanden ist und schön verteilt werden soll.

column-count: 3; column-gap: 24px;
